# pixiublog

a blog that use golang, base on beego,layui

## tech

+ beego
+ bee
+ layui
+ bootstrap
+ PPGO_JOB

## install

- go get https://github.com/leonardodacn/pixiublog
- create mysql database and import pixiublog.sql to the database
- modify config/app.conf
- run `go build`
- `./run.sh start|stop`
